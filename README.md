# 🚀 Portfolio Website

A **modern and responsive** portfolio website built with the **MERN stack** (MongoDB, Express, React, Node.js).

## 🛠️ Tech Stack
- **Frontend**: React + Vite
- **Backend**: Node.js + Express
- **Database**: MongoDB + Mongoose
- **Styling**: Custom CSS

---

## ⚡ Features
- **Dynamic Project Showcase** ✨
- **Contact Form with Email Integration 📧**
- **Responsive Design 📱**

---

## 📦 Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Aabi0207/Portfolio-Website.git
   cd Portfolio-Website
   ```

2. **Backend Setup**
   ```bash
   cd backend
   npm install
   cp .env.example .env
   npm run seed
   npm run dev
   ```

3. **Frontend Setup**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

---

## 🌐 Usage
- **Frontend**: `http://localhost:5173`
- **Backend API**: `http://localhost:8000/api`

Make sure MongoDB is running locally, or set `MONGODB_URI` in `backend/.env`.
